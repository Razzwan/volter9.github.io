Как со мной связатся

Со мной можно связатся по почте, Твиттеру или Скайпу (я использую скайп как IM). Для того чтобы получить мою почту или Скайп, вам прийдется запустить данный PHP код:

/**
 * Скопируйте и вставьте весь код для получения
 * мыла и Скайпа. 
 */
$base = 'aWJ5Z3JlOTI1QHR6bnZ5LnBieg==';
$mail = base64_decode($base);
$mail = str_rot13($mail); 

echo "Моя почта: $mail";

/**
 * Если не нужен скайп, выполнять код ниже нету смысла
 */
$skype = substr($mail, 0, 7);
$skype = str_replace('9', '_9', $skype);

echo "Мой Скайп: $skype";

Скрипт очень простой, достаточно его запустить в интерактивной консоле (php -a или psysh) чтобы получить скайп/почту.
