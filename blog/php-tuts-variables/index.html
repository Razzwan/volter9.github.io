Привет!

Я решил ради эксперимента (и повышения своих навыков) написать небольшой вводный курс по PHP. Это вводный туториал.

PHP — это язык программирования заточенный под веб. PHP сильно отличается от других языков тем что изначально создавался как шаблонизатор и по сей день имеет возможности шаблонизатора.

Любой PHP код будет работать только внутри <?php ... ?> тэгов. Эти тэги называются PHP тэги. Если в файле только PHP код, то можно упустить закрывающийся тэг.
